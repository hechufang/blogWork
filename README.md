# 为什么会创建这个项目呢？
* 最近找工作压力很大，2015年开始参加工作，到现在已经工作四年了，项目经验也很丰富，但是对于算法、数据结构、网络、以及iOS基础知识的掌握还是有所欠缺，希望能通过这个项目学习、提升一下自己的能力。
* 另外2015-2017年，在简书、知乎都写过一些文章，后来被封了账号，所以这次吸取教训，在github也备份一下。
